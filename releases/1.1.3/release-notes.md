# QuickFocus 1.1.3

## Highlights

- Workspace popup keyboard navigation is now column-aware (`Up/Down` inside current column, `Left/Right` between adjacent visible columns).
- `Enter` on a different workspace keeps popup navigation active; `Enter` on the already active workspace closes popup and restores focus to the top input.
- `Esc` in workspace popup closes it and restores focus to the top input.
- Workspace popup is now organized in 4 columns (`Workspaces`, `Dynamic`, `TAGs`, `References` + `Saved Searches`) with empty columns/categories automatically hidden.
- Opening a `Saved Search` workspace now clears the top active filter to avoid accidental double filtering.
