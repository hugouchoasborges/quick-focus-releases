# QuickFocus 1.0.30

## Highlights

- Added `Duplicate Task` / `Duplicar Tarefa` to the task context menu, reusing the existing `Ctrl + D` behavior.
- Refined task context menu layout with separators, compact shortcut rendering, and `Move to Workspace` repositioned to the final section.
- Updated `Copy as Text` output formatting to a compact structure with child bullets and adjusted trailing separator behavior.
- Added theme-specific Pomodoro icons (black for light theme, white for dark theme) and aligned rendered icon dimensions with the new asset ratio.

## Fixes

- Fixed startup failure when themed Pomodoro icon files were missing from output/package by including new icon assets in publish content.
