# QuickFocus 1.0.23

- Reusable confirmation popup flow standardized for destructive actions and applied to `Remove Workspace`.
- Top input now supports `Modifier + Enter` to create a task at the top of the list.
- TAG settings now assign a random color to new TAGs and auto-scroll to the newly added item.
- Fixed top-input create-at-top flow to preserve typed task text.
