# QuickFocus 1.0.36

## Highlights

- Fixed ambient `Office` sound to use a dedicated source file instead of mirroring `Cafe`.
- Fixed workspace popup counters to ignore hidden upcoming tasks configured with `Show task only on reminder day`.
- Fixed `Ctrl+K`, `Ctrl+L`, and `Ctrl+N` while editing child tasks to target the child task.
- Added `Tick Sound` toggle to the Pomodoro start popup, synchronized with settings.
