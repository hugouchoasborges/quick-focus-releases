# QuickFocus 1.0.24

## Highlights

- Improved keyboard navigation while editing task text: Up/Down now follows visible tasks, including expanded subtasks, and stays anchored at list limits.
- Extended two-click edit behavior to subtasks with a clear first-click visual arm indicator.
- Clicking a subtask without Ctrl/Shift now clears selected parent-task context.
- Pressing Delete now removes a selected subtask when top-level multi-selection is not active.
- Fixed subtle text shift when entering edit mode so the task text stays visually stable.

## Notes

- Technical release focused on interaction polish and editing consistency.
