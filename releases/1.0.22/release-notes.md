# QuickFocus 1.0.22

- Added `Dynamic Workspaces` / `Workspaces Dinamicos` setting in `TAGs` (enabled by default) to control dynamic workspace creation/visibility in the workspace popup.
- Workspace popup count for normal workspaces now includes only parent tasks (subtasks excluded).
- Active filter chip in the top-right now turns red when the active filter has zero results in the current workspace.
