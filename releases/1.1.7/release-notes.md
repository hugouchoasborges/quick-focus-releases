# QuickFocus 1.1.7

## Highlights

- Drag-and-drop em lote agora move todo o conjunto selecionado quando o arraste comeca em uma tarefa da selecao multipla.
- Drop na popup de workspaces agora suporta o mesmo arraste em lote para mover tarefas entre workspaces em uma unica operacao.
- Tarefas filhas selecionadas agora usam borda esverdeada (`ChildAccentBrush`) para diferenciar visualmente das tarefas nao-filhas.
