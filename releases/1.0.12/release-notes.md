# QuickFocus 1.0.12

- Added a high-contrast main window border by theme (`black` in light mode and `white` in dark mode), improving edge visibility.
- Simplified task row actions by removing move controls (`up`, `down`, `move to workspace`) and keeping delete (`X`) as the rightmost action.
- Fixed icon theme refresh so all theme-dependent icons are reapplied correctly when toggling between light and dark themes.
- Corrected `Sync Now` icon alignment/rotation center and improved glyph clarity.
