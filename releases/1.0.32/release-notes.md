# QuickFocus 1.0.32

## Highlights

- Fixed recurring parent completion when parent is auto-completed from child completion: next recurrence is now scheduled and the completed parent reminder is cleared.
- Updated top input shortcuts: `Shift + Enter` creates at top, and `Ctrl + Enter` creates task and focuses the created task text at the end.
- While editing a task, `Shift + Delete` now removes the current task.
- Updated the shortcuts/help window and localized labels to reflect the new keyboard behavior.
