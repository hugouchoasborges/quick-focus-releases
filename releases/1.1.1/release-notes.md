# QuickFocus 1.1.1

## Highlights

- Subtasks now support full multi-selection flow (mouse and keyboard) just like regular tasks.
- Context menu is now available on subtasks, including right-click access and batch TAG operations.
- Moving a child task alone to another workspace now promotes it to a normal task in destination.
- Subtasks now preserve and receive TAGs, including `Auto TAGs`; forced child TAG cleanup was removed.
